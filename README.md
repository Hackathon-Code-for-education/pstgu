# ABITU

## Стек

- PHP 8.0
- Laravel 10.0
- Docker
- Bootstrap 5.0

## Реализовано (20 апр, 1/3)

- Главная страница
- Дизайн проекта
- Страница авторизации
- Страница о проекте (about)
- Страница ответов на вопросы (faq)

## Запуск

1. Форкнуть репозиторий
2. Запустить Docker
3. Запустить контейнер
4. Открыть localhost для доступа
