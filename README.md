# ABITU

## Запуск

1. Скопировать репозиторий
2. Запустить Docker
3. Выполнить команду 'cd pstgu && ./vendor/bin/sail up'
4. Открыть http://localhost

## Стек

- PHP 8.0
- Laravel 11.0
- Docker
- Bootstrap 5.0
- MySQL
- Хэширование пароля
- Панорамы ВУЗа
